How to Install & Run

1. Prerequisites

Ensure you have Python installed on your system. You also need Visual Studio Code (VS Code).

2. Setup VS Code Python Extension

Open VS Code, go to the Extensions view (Ctrl+Shift+X), search for "Python" by Microsoft, and click Install.

3. Install Dependencies

Open the terminal in the project folder and run:

pip install -r requirements.txt

4. Run the Application

Start the server/application by running:

python main.py
